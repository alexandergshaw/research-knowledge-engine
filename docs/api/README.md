API documentation for the Next.js API routes in `apps/web`.

## Planned Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/api/search` | Full-text search over indexed sources |
| `GET` | `/api/sources` | List paginated sources |
| `GET` | `/api/sources/:id` | Get a single source |
| `POST` | `/api/reports` | Generate a research report |
| `GET` | `/api/reports/:id` | Retrieve a generated report |
| `GET` | `/api/feeds` | List configured RSS feeds |
| `POST` | `/api/jobs` | Enqueue a background job |
